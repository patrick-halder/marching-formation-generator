# Marschaufstellung-Generator

Webanwendung zur Generierung von Marschformationen für Blasorchester.

Spieleranzahl eingeben → Formation berechnen → per Drag-and-Drop anpassen → als PNG exportieren.

## Lokal starten

```bash
cd public
python -m http.server 8080
```

Dann öffnen: http://localhost:8080

## Deployment

Push auf `main` → GitHub Actions deployt automatisch auf GitHub Pages.
